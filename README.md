# Excel Sales Dashboard Project

## Project Overview
This project is an interactiveExcel Dashboard created to analyze and visualize sales data effectively.  
It helps in understanding region-wise and product-wise performance, trends, and key metrics for better decision-making.

## Features
- **Interactive Dashboard Sheet**: Displays charts and summaries for easy visualization.
- **Pivot Tables**: For analyzing sales data by **Region** and **Product**.
- **Charts Included**:
  - Region-wise Sales (Column Chart)
  - Product-wise Sales (Bar/Column Chart)
  - Overall Sales Performance (Summary Metrics)

## File Structure
- `RawData.xlsx` – Original sales data
- `Analysis` – Pivot tables and calculations
- `Dashboard` – Final dashboard with charts and KPIs

## How to Use
1. Open the Excel file.
2. Go to the Dashboard sheet.
3. View charts and summaries.

## Skills/Technologies Used
- Microsoft Excel
- Pivot Tables
- Data Visualization (Charts)
- Dashboard Design

## Outcome
- Quickly analyze sales performance by region and product.
- Gain insights for better business decisions.
